Principales Etiquetas Utilizadas en esta Página.



HTML no es un lenguaje de programación, eso lo sabe todo el mundo, ya que el archivo utiliza un sistemas de etiquetas para dar instrucciones de que debe mostrar nuestro navegador, esto sin tener que interpretarlo o compilarlo. Aquí un listado de algunas etiquetas que, en menor o mayor medida, me han servido para crear esta página web.

  • < html >

    Es la etiqueta más importante de todas ya que es la que inicia el archivo html, por consecuente tambien las demas etiquetas.

  • < head >

    Aquí adentro van todos los metadatos, conexiones a demas archivos en la carpeta(CSS,bootstrap,JS) y el título de la página.

  • < body >

    Dentro de esta etiqueta entra todo el contenido visible en la página web. Le coloque onclick para que reproduzca audio cuando el usuario interraccione con la página por primera vez.

  • < div >

    Me permitió el separar por divisiones los diferentes contenedores de la página para que así no tenga inconvenientes al momento de diseñar.

  • < br >

    Su proposito es generar un salto de línea, permitiendo que el contenido visible esté mejor estructurado.

  • < style. >

    La utilice para modificar el estilo de la página sin necesidad de modificar el archivo de bootstrap ya anexado.

  • < script. >

    Me permitió inyectar código de JS al archivo html sin necesidad de conectar otro archivo. La página se vuelve más interactiva a causa de eso.
